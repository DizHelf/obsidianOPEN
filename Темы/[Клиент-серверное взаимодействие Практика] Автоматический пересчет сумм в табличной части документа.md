## Длинная версия кода

```js
&НаКлиенте

Процедура ПолучениеСуммыЦенТоваров()

	ТекущиеДанные = Элементы.Товары.ТекущиеДанные;
	Количество = ТекущиеДанные.Количество;
	Цена = ТекущиеДанные.Цена;
	
	Если Количество > 0 и Цена > 0 Тогда
		
		Индекс = ТекущиеДанные.НомерСтроки - 1;
		Объект.Товары[Индекс].Сумма =  Количество * Цена;
	
	КонецЕсли;

КонецПроцедуры // ()
```
## Короткая версия кода

По факту условие не нужно, так как пустая строка переводится к 0, а ещё **ТекущиеДанные** являются ссылкой на объект поэтому изменение полей в нём приведёт к изменению полей в объекте. 

```js
&НаКлиенте
Процедура ПолучениеСуммыЦенТоваров()

	ТекущиеДанные = Элементы.Товары.ТекущиеДанные;
	
	ТекущиеДанные.Сумма = ТекущиеДанные.Количество * ТекущиеДанные.Цена;

КонецПроцедуры // ()
```
